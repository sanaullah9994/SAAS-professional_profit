# MVP roadmap

1. **Foundation:** included in this scaffold.
2. **Tenant onboarding:** create organization/workspace after signup; owner/admin/analyst guards.
3. **Seller Central:** LWA callback, KMS token encryption, orders, financial events, refunds, 90-day initial sync.
4. **Amazon Ads:** OAuth, profile discovery, sponsored ads reports, SKU mapping, unattributed spend.
5. **Profit engine:** reconciliation, dated COGS, day/week/month reporting, operating-cost allocation.
6. **Production:** Vercel, ECS Fargate, Neon, Upstash, S3, Sentry, CloudWatch, alarms and runbooks.
